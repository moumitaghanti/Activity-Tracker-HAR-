A Human Activity Recognition Project

Objective

The purpose of this project is to see if we could take an existing idea (health trackers) and completely automate to provide an overall summary of your daily acitivities. Current devices require you to set it all up to record, but what if you didn't have to?

Our idea is create a program that would use a smartphone/mobie device to recognize and predict a specific activity from data collected using an accelerometer [and gyroscope].


Data Collection

As fully vested collaborators, we thought it would be interesting to include ourselves as participants of our own study. After installing an accelerometer app onto our smartphones, the data collection began. From this data, we would train and test our models for accuracy.
